<게시글 상세>

1. url 선정 : /board/detail/게시글 번호

2. 컨트롤러
 - 메소드 추가
  - value : /board/detail/게시글번호 => @PathVariable
  - method : get
  - viewName : /board/detail

3. detail.jsp 추가

4. 컨트롤러
 - 2번 메소드에 작업
 - 게시글 번호를 가져옴(테스트)
 - 서비스에게 게시글 번호를 주면서 게시글 정보를 가져오라고 요청
 - 가져온 게시글을 화면에 전달

5. 서비스/임플
 - 메소드 추가
   - 다오에게 게시글 번호를 주면서 조회수를 증가하라고 요청
   - 다오에게 게시글 번호를 주고 게시글 정보를 가져오라고 요청 한 후 리턴

6. 다오/매퍼
 - 메소드 추가 및 쿼리 구현
   - update문을 이용하여 조회수 증가하는 쿼리 구현
   - select문을 이용하여 게시글 정보를 가져오는 쿼리 구현

7. detail.jsp
 - 서버에서 보내준 게시글을 화면에 구성
